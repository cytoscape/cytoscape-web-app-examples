# Per-App Design Documents

Each subdirectory contains the design document for one example app.

| App | Purpose | Design Doc |
|-----|---------|------------|
| [hello-world/](hello-world/) | Minimal panel-only example | [design doc](hello-world/README.md) |
| [network-workflows/](network-workflows/) | Advanced network creation and external integration example | [design doc](network-workflows/README.md) |
| [simple-menu/](simple-menu/) | Minimal menu item example | [design doc](simple-menu/README.md) |
| [simple-panel/](simple-panel/) | Minimal panel example | [design doc](simple-panel/README.md) |
| [project-template/](project-template/) | Boilerplate for new apps | [design doc](project-template/README.md) |

## Template

When adding a new app, copy the structure from `project-template/README.md` as a starting point.
