# network-workflows App

- App ID: `networkWorkflows`
- Default entry point (local): `networkWorkflows@http://localhost:7000/remoteEntry.js`

## Description

This app collects the non-minimal workflow examples that used to live in
`hello-world`.

It demonstrates:

1. Creating a sample network from a menu item
2. Importing a network from a remote CX2 file
3. Receiving CX2 data from Jupyter Lab via `postMessage`
